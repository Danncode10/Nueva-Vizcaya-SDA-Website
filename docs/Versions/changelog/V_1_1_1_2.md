#### Detailed Explanations
- **Technical Details**: Set up Python virtual environment using venv, installed the following libraries:
  - __fastapi__: Modern web framework for building REST APIs with automatic OpenAPI docs, async support, and type validation.
  - __uvicorn__: ASGI server to run FastAPI applications, handles HTTP requests efficiently.
  - __sqlalchemy__: Object-Relational Mapper (ORM) for interacting with databases using Python objects instead of raw SQL.
  - __psycopg2-binary__: PostgreSQL database adapter, allows Python to connect and query PostgreSQL.
  - __boto3__: AWS SDK for Python, used to interact with AWS services like Cognito for authentication.
  - __python-jose[cryptography]__: Library for JSON Web Tokens (JWT) encoding/decoding with cryptographic signing for secure tokens.
  - __pydantic[email]__: Data validation library with built-in email validation, ensures data integrity in API requests/responses.
  Ran `pip freeze > requirements.txt` to capture exact package versions for reproducibility.
